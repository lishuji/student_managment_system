# Java教务管理系统

## 项目介绍
- 基于Spring的教务管理系统、学生管理系统、课表查询系统

## 功能介绍
- 登录、注册、修改密码、退出登录
- 教室、班级、院系、地点
- 教工、学生、用户
- 课程、课程表、课程查询
- 排课、排课查询

## 使用到的技术
```
Spring Boot
SpringMVC
MyBatis
Maven
MySQL
thymeleaf
mybatis-plus
druid
Lombok
Hutool
H-ui
Timetables.js
```

# 使用说明

- SMS目录下的数据库文件database.sql导入MySQL数据库，运行程序即可
- 后台登录界面为 admin.aspx ，默认账号：admin 密码：admin
